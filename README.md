# Java GUI To Do List Application

---
# Overview
This project is a Java GUI application that allows users to manage a list of items. Users can add, remove, and view tasks/items
through a graphical interface. The application is built using Java Swing for the GUI.

---

# Features
- Add new items to the list
- Remove selected items
- Display all items in a clear list view
- Simple and intuitive graphical interface
- Lightweight and easy to run

---
# Technologies Used
- Java (JDK 8 or above)
- Swing GUI Toolkit
- NetBeans IDE (or any Java IDE)

---

# How to Run
- Open the project in NetBeans (or your preferred IDE)
- Build the project
- Run the Main.java file
- The GUI window will open, allowing you to interact with the list

---
# User Guide
- Add Item: Type the item in the input field and click Add
- Remove Item: Select an item in the list and click Remove
- View List: All items appear in the main list panel
